    安装Java

    1, 更新包

    yum update

    2, 检测是否已安装

    java -version

    3, 去官网下载Java8 (rpm), 找到下载链接

    http://www.oracle.com/technetwork/java/javase/downloads/index.html

    Mac 下点击下载图标,开始下载后取消,右键获取下载链接

    http://download.oracle.com/otn-pub/java/jdk/8u181-b13/96a7b8442fe848ef90c96a2fad6ed6d1/jdk-8u181-linux-x64.rpm?AuthParam=1533226087_a303b41d997c044de6470da5b9a3d806

    4, 下载

    wget http://xxxx

    5, 重命名下载的文件

    mv xxx xxx.rpm

    6, 安装

    rpm -ivh jdk-8u181-linux-x64.rpm

    7, 检测是否安装成功, 确认Java安装路径: /usr/java/

    java -version

    8, 设置java环境变量

    vi ~/.bash_profile

    编辑以下内容

    使配置生效

    source ~/.bash_profile

    9, 确认环境变量是否设置成功

    echo $JAVA_HOME
